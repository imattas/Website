# picoCTF 2026 Writeups

Full writeups (with embedded solution scripts) for all 70 challenges from **picoCTF 2026** (Event 79).

**Total Points: 14,500 / 14,500** | **70 / 70 Challenges Solved** | **Author: Imattas aka Zemi**

## Writeup Format

Each challenge is a single Markdown file following a consistent template:

- **Header** — Category, Difficulty, Points, Author
- **Challenge Description** — the original prompt
- **Recon / Initial Analysis** — first look at the target
- **Vulnerability / Observation** — the bug class or key insight
- **Exploitation / Solution** — step-by-step walkthrough
- **Full Exploit Script** — the complete solve script
- **Key Takeaways** & **References**

Difficulty is derived from points: **Easy** (50–100), **Medium** (150–300), **Hard** (400+).
